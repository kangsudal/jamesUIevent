# event3ui

MainState -> _eventWidth -> Row()
->Left()
->Right()
로 구성해서 만들어보았습니다 :)

MainState에 bool alreadyClicked라는 변수로 하트아이콘 버튼 클릭이벤트 상태를 관리해주었고
Listview, 레이아웃, 정렬같은거 머리랑 눈으로만 따라봤지 제대로 연습할기회가 부족했는데 이번 이벤트 도전을 통해서 좋은시간을 가졌습니다
